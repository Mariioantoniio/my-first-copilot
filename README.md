# 🛡️ InfraSec Copilot

Um copiloto especializado em Infraestrutura, Segurança da Informação, Cloud, Redes, Governança e Operações.

O objetivo deste projeto é demonstrar como criar diferentes modos de atuação para um assistente de IA voltado para ambientes corporativos.

---

# Modos Disponíveis

## 🔍 Analyze

Utilizado para análise e troubleshooting.

Ideal para:

* Logs
* Alertas
* Incidentes
* Configurações
* Diagnóstico de falhas

Exemplos:

* Analise este log do Wazuh.
* Explique este erro do Linux.
* Investigue este alerta de segurança.

---

## 🛠️ Execute

Utilizado para implementar mudanças.

Ideal para:

* Scripts
* Hardening
* Configurações
* Firewalls
* Servidores

Exemplos:

* Configure retenção de logs.
* Crie um script de backup.
* Ajuste esta regra do pfSense.

---

## 📋 Plan

Utilizado para planejar mudanças.

Ideal para:

* Migrações
* Projetos
* Roadmaps
* Melhorias de ambiente

Exemplos:

* Planeje uma migração de firewall.
* Crie um roadmap de hardening.
* Planeje a implantação de um SIEM.

---

## 🤖 Operator

Modo operacional avançado.

Capaz de:

* Trabalhar em múltiplas etapas
* Correlacionar informações
* Definir sequência de execução
* Conduzir projetos complexos

Exemplos:

* Estruture uma arquitetura completa de SIEM.
* Padronize um ambiente seguindo CIS.

---

## 🎓 Learn

Modo de aprendizado.

Ideal para:

* Estudo guiado
* Capacitação técnica
* Explicação de conceitos
* Simulações

Exemplos:

* Ensine Active Directory.
* Explique VLANs.
* Ensine Wazuh do básico ao avançado.

---

## 🛡️ Audit

Modo de auditoria e compliance.

Baseado em:

* ISO 27001
* CIS Controls
* NIST
* MITRE ATT&CK

Exemplos:

* Audite este servidor Linux.
* Avalie esta configuração do Wazuh.
* Verifique aderência à ISO 27001.

---
